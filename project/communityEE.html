  • 프로젝트 소개

    Community EE는 직장생활에 대한 애환들을 서로 소통하며, 직무별 정보를 공유할 수있는 커뮤니티 사이트입니다.
    신촌 비트캠프 JAVA 143기 비전공 교육생 4명이 모여 제작하였습니다.

    * 커뮤니티 사이트를 선택한 이유
    6개월간의 국비교육과정에서 배운 내용들을 비롯하여 각각의 게시판에는 서로 다른 기능을 적용가능하므로, 기술 스펙트럼을 최대한 전개할 수 있는 주제라고 생각하여 프로젝트 주제로 커뮤니티를 선택

    * 팀원구성
    도희정, 이혜인, 김민영, 박찬경

  • 개발기간

    2020.01.28 ~ 2020.04.22
